# Style Guide

## Color Palette

- `#FFFFFF` - primary background
- `#131313` - primary text
- `#A8DCAB` - soft highlight accents
- `#519755` - primary accent (links and post title emphasis)
- `#DBAAA7` - secondary accent (borders and warm highlights)
- `#BE91BE` - tertiary accent (section headings and subtle pops)

## Current Usage Notes

- Blog detail post title (`.single-post h1`) uses `#519755` with a `#DBAAA7` underline accent.
- Playlist heading (`.playlist-block h2`) uses `#BE91BE`.
- Homepage cards continue rotating accent-left borders using `#A8DCAB`, `#DBAAA7`, and `#BE91BE`.
